Rede Social Django - API de Teste

Esta aplicação implementa um modelo de rede social com base nos predicados Prolog fornecidos.

Endpoints Disponíveis

Usuários que curtiram reels

Equivalente à regra Prolog: curtiu_reel(nome)

URL: /api/usuarios-que-curtiram-reels/

Usuários que comentaram em postagens

Equivalente à regra Prolog: comentou_em_postagem(nome, texto)

URL: /api/usuarios-que-comentaram-em-postagens/

Quem segue quem

Equivalente à regra Prolog: segue(nome1, nome2)

URL: /api/quem-segue-quem/

Comentários em reels

Equivalente à regra Prolog: comentarios_em_reels(nome, texto)

URL: /api/comentarios-em-reels/

Filtrar por usuário: /api/comentarios-em-reels/{nome_usuario}/

Estatísticas de usuário

Retorna estatísticas detalhadas de um usuário específico

URL: /api/estatisticas-usuario/{username}/

Administração

Acesse o painel de administração em /admin/ para gerenciar os dados.
